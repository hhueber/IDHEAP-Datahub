# Documentation

## Choropleth Map Service

### Definition

**EN :**  
A **choropleth map** is a thematic map where geographic areas
(for example municipalities, regions, or countries) are colored according to
a data value (statistical metric, survey result, numerical score, etc.).
Each area receives a color based on its associated value, allowing quick
visualization of spatial patterns in the data.

### Why this term appears in the project

The term **choropleth** is used in some file or component names in this project
because they handle the logic responsible for generating or displaying these
colored geographic data maps.
